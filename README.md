# EEST Nº 2, GDAD.

## Esta es la pagina web oficial de la EEST Nº 2 de San Nicolas.

## Instalacion: 

```bash
git clone https://github.com/tu-usuario/nombre-del-proyecto.git
``` 

```bash
cd nombre-del-proyecto
``` 

```bash
npm install
``` 
## Uso

```bash 
npm run dev
```
Posteriormente abrir en el navegador: http://localhost:3000
